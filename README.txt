Приложение TPclient предназначено для работы в устройстве
конроллера печати на базе raspberry pi3B+ для струйного принтера.
TPclient подключается к серверу печати и производит обмен
данными по протоколу TCP/IP. Кроме того, приложение управляет
световой сигнализацией и получает подтверждение от оператора о
том, что документ действительно напечатан.

Проект реализован в среде VS C++ в режиме удалённой отладки.
распространяется в виде бинарного файла TPclient.out
